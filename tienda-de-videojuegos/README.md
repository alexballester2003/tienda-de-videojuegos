# Proyecto Web - Pre Entrega

Este proyecto consiste en una página web desarrollada con HTML y CSS.

## Contenido del proyecto

- Estructura semántica HTML.
- Navegación interna.
- Sección de productos con Flexbox.
- Sección de reseñas con Grid.
- Formulario de contacto usando Formspree.
- Diseño responsive mediante Media Queries.
- Integración de contenido multimedia.

## Tecnologías utilizadas

- HTML5
- CSS3
- Google Fonts
- Flexbox
- CSS Grid
